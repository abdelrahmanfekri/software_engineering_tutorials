# Causal Machine Learning Tutorial: From Inference to Discovery

## Overview
Comprehensive guide to causal inference and causal machine learning. Learn to move beyond correlation to understand causation, critical for scientific reasoning and robust AI systems.

## Table of Contents

1. **[Causality Foundations](01-causality-foundations.md)** - Causal graphs, DAGs, do-calculus, structural causal models, intervention vs observation
2. **[Causal Discovery](02-causal-discovery.md)** - PC algorithm, FCI, LiNGAM, constraint-based and score-based methods, structure learning
3. **[Causal Inference Methods](03-causal-inference.md)** - Propensity scores, matching, IV, regression discontinuity, DiD, synthetic controls
4. **[Counterfactual Reasoning](04-counterfactual-reasoning.md)** - Counterfactuals, potential outcomes, treatment effects, mediation analysis
5. **[Causal Representation Learning](05-causal-representation-learning.md)** - Learning causal variables, identifiability, disentanglement, causal feature learning
6. **[Causal RL](06-causal-rl.md)** - Causal world models, confounded RL, off-policy evaluation, causal imitation learning
7. **[Applications](07-causal-applications.md)** - Medicine, policy, business, A/B testing, recommendation systems, fairness

## Prerequisites
- Statistics and probability
- Machine learning basics
- Graph theory basics

## Tools: CausalML, DoWhy, ananke, pgmpy, PyMC

## Time: 8-10 weeks
