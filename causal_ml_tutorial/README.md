# Causal Machine Learning Tutorial

## Description
Learn causal inference and causal machine learning. Move beyond correlation to understand true causal relationships, essential for scientific AI and robust decision-making.

## What You'll Learn
- Causal graphs and do-calculus
- Causal discovery algorithms
- Causal inference (propensity scores, IV, DiD)
- Counterfactual reasoning
- Causal representation learning
- Causal RL

## Prerequisites
- Statistics and probability
- ML basics
- Graph theory
