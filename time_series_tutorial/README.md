# Time Series and Forecasting Tutorial

## Description
Master time series analysis and forecasting from classical statistical methods to modern deep learning architectures. Essential for finance, supply chain, energy, and more.

## What You'll Learn
- Classical methods (ARIMA, exponential smoothing)
- Neural forecasting (LSTM, TCN)
- Transformer-based forecasting
- Probabilistic forecasting
- Anomaly detection
- Multivariate time series

## Prerequisites
- Statistics
- Time series basics
- Deep learning

7 modules from fundamentals to advanced
