# Reinforcement Learning Tutorial: From Fundamentals to Advanced Algorithms

## Overview
Comprehensive RL tutorial covering tabular methods, deep RL, policy gradients, actor-critic, multi-agent RL, and applications to LLMs (RLHF). Build practical RL systems from scratch.

## Table of Contents

1. **[RL Foundations and MDPs](01-foundations-mdps.md)** - MDPs, Bellman equations, value functions, policies, RL problem formulation
2. **[Tabular Methods](02-tabular-methods.md)** - Q-learning, SARSA, TD learning, eligibility traces, convergence analysis
3. **[Function Approximation](03-function-approximation.md)** - DQN, Double DQN, Dueling DQN, Rainbow, prioritized experience replay
4. **[Policy Gradient Methods](04-policy-gradients.md)** - REINFORCE, REINFORCE with baseline, variance reduction, natural policy gradients
5. **[Actor-Critic Algorithms](05-actor-critic.md)** - A2C, A3C, PPO, TRPO, SAC, TD3, implementation and tuning
6. **[Model-Based RL](06-model-based-rl.md)** - World models, Dyna, MuZero, AlphaZero, planning with learned models
7. **[Multi-Agent RL](07-multi-agent-rl.md)** - QMIX, MADDPG, game theory, cooperation, competition, communication
8. **[Inverse RL](08-inverse-rl.md)** - IRL theory, Maximum Entropy IRL, GAIL, learning from demonstrations
9. **[RLHF Deep Dive](09-rlhf-deep-dive.md)** - Reward modeling, PPO for LLMs, Constitutional AI, alignment techniques
10. **[Offline RL](10-offline-rl.md)** - Conservative Q-Learning, IQL, behavior cloning, offline-to-online
11. **[RL for LLMs](11-rl-for-llms.md)** - Fine-tuning with RL, instruction following, reasoning with RL, self-improvement

## Prerequisites
- Python, NumPy
- Basic ML/DL
- Probability theory
- Calculus and optimization

## Tools: Gym, Stable-Baselines3, Ray RLlib, PyTorch

## Time: 10-12 weeks
