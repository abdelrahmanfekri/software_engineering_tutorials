# Reinforcement Learning Tutorial

## Description
Master reinforcement learning from tabular methods to state-of-the-art deep RL algorithms. Includes comprehensive coverage of RLHF for LLM alignment and multi-agent systems.

## What You'll Learn
- RL fundamentals (MDPs, Bellman equations)
- Tabular methods (Q-learning, SARSA)
- Deep RL (DQN, PPO, SAC)
- Multi-agent RL
- RLHF and LLM alignment
- Model-based RL

## Prerequisites
- Python programming
- ML/DL basics
- Probability and calculus

11 modules from basics to LLM applications
