# Graph Neural Networks Tutorial

## Description
Master GNNs and geometric deep learning for graph-structured data. Applications in molecules, social networks, knowledge graphs, and 3D geometry.

## What You'll Learn
- Graph theory and representations
- Graph convolutions (GCN, GAT, GraphSAGE)
- Graph generation
- Molecular ML and drug discovery
- Geometric deep learning
- Real-world applications

## Prerequisites
- Graph theory basics
- Deep learning
- Linear algebra
