# Graph Neural Networks and Geometric Deep Learning

## Overview
Comprehensive guide to GNNs for learning on graph-structured data. Applications in social networks, molecules, knowledge graphs, and 3D geometry.

## Table of Contents

1. **[Graph Fundamentals](01-graph-fundamentals.md)** - Graph theory, representations, algorithms, spectral graph theory, graph signal processing
2. **[Graph Convolutions](02-graph-convolutions.md)** - GCN, GraphSAGE, GAT, GIN, message passing, pooling, implementation with PyG
3. **[Graph Generation](03-graph-generation.md)** - GraphRNN, GRAN, flow-based models, diffusion for graphs, molecular generation
4. **[Molecular ML](04-molecular-ml.md)** - Drug discovery, property prediction, SMILES, 3D molecular graphs, SchNet, DimeNet
5. **[Geometric Deep Learning](05-geometric-deep-learning.md)** - Equivariance, symmetries, geometric priors, point clouds, meshes
6. **[Applications](06-gnn-applications.md)** - Social networks, recommender systems, traffic prediction, biology, physics

## Prerequisites
- Graph theory basics
- Deep learning
- Linear algebra

## Tools: PyTorch Geometric, DGL, NetworkX, RDKit
