# Edge AI and TinyML Tutorial

## Overview
Learn to deploy AI models on edge devices and embedded systems. Essential for IoT, mobile applications, and privacy-preserving AI.

## Table of Contents

1. **[Edge Constraints](01-edge-constraints.md)** - Memory, compute, power limitations, real-time requirements, optimization strategies
2. **[Model Compression](02-model-compression.md)** - Quantization (INT8, INT4), pruning, knowledge distillation, neural architecture search for efficiency
3. **[Mobile Deployment](03-mobile-deployment.md)** - TensorFlow Lite, Core ML, ONNX Runtime Mobile, optimization, profiling, app integration
4. **[Embedded Systems](04-embedded-systems.md)** - Microcontrollers, TensorFlow Lite Micro, MCU deployment, Arduino, ESP32, STM32
5. **[Federated Learning](05-federated-learning.md)** - On-device training, privacy preservation, communication efficiency, FedAvg, applications

## Prerequisites
- ML fundamentals
- Mobile/embedded basics helpful
- C/C++ for embedded

## Tools: TF Lite, Core ML, ONNX, Edge TPU, TinyML

## Time: 4-6 weeks
