# AI for Code Tutorial

## Description
Learn to build AI systems for code generation, understanding, bug detection, and automated development. Covers modern LLMs for code, program synthesis, and practical AI-assisted development workflows.

## What You'll Learn
- Code understanding (ASTs, CFG, embeddings)
- Code generation with LLMs
- Program synthesis techniques
- Bug detection and code review automation
- Test generation
- AI-assisted development tools

## Prerequisites
- Strong programming skills
- Basic compiler knowledge
- ML/NLP fundamentals
